# Vert.x Redis Client examples

Here you will find examples demonstrating the usage of the Vert.x Redis Client.

# Project setup

To use es4x your own project use the following project as a template:

{
  "name": "core-examples",
  "version": "1.0.0",
  "private": true,
  "license": "ISC",
  "dependencies": {
    "create-es4x-app": "latest",
    "@vertx/core": "latest",
    "@vertx/redis-client": "latest"
  },
  "scripts": {
    "postinstall": "es4x install"
  }
}

To run this example, you need a running Redis instance. Once running, you can configure the verticles with the redis host:

/// <reference types="es4x" />
// @ts-check

import { Redis, Command, Request } from "@vertx/redis-client";

// Create the redis client
var client = Redis.createClient(vertx, "redis://127.0.0.1:6379");

client.send(Request.cmd(Command.SET).arg("key").arg("value"), onSet => {
  if (onSet.succeeded()) {
    console.log("key stored");
    client.send(Request.cmd(Command.GET).arg("key"), onGet => {
      console.log("Retrieved value: " + onGet.result().toString());
    });
  } else {
    console.trace(onSet.cause());
  }
});

By default it uses 127.0.0.1 as host. The post is set to 6379.
